Meus Projetos 💗

Pet Acolhe

Pet Acolhe é um projeto web funcional desenvolvido com Laravel para simulação de uma plataforma de adoção de pets. No sistema, o usuário pode se cadastrar, visualizar animais disponíveis, solicitar adoção, cadastrar pets para doação e realizar doações em dinheiro, com aprovação administrativa das adoções.
